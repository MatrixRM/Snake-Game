# 🐍 Snake Game

Um jogo da cobra clássico implementado com **HTML5**, **CSS3** e **JavaScript vanilla**.

## 🎮 Características

- ✨ Interface moderna e responsiva
- 🎯 Sistema de pontuação com melhor score
- ⚡ Dificuldade progressiva (aumenta a velocidade)
- 🎹 Controles via teclado (Setas ou WASD)
- 💾 Persistência de dados com localStorage
- 📱 Design mobile-friendly
- 🌈 Visual atraente com efeitos

## 🚀 Como Usar

1. Abra `index.html` em um navegador web
2. Clique em **"Iniciar Jogo"** para começar
3. Use as **setas do teclado** ou **WASD** para controlar a cobra
4. Coma a comida vermelha para ganhar pontos
5. Evite as paredes e colidir com você mesmo

## 🎮 Controles

| Controle | Ação |
|----------|------|
| ⬆️ / W | Mover para cima |
| ⬇️ / S | Mover para baixo |
| ⬅️ / A | Mover para esquerda |
| ➡️ / D | Mover para direita |
| Espaço | Pausar/Retomar |

## 📊 Pontuação

- Cada comida = **+10 pontos**
- Quanto mais você come, mais rápido o jogo fica!
- Seu melhor score é salvo automaticamente
